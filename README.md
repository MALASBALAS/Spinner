# Preferencias

## ¿Qué hace?
La aplicación obtiene datos de un recurso XML, y pinta estos datos en el layout mediante el uso de Spinner, LIstView y GridView.

## Funcionamiento
1. La actividad entra en el recurso XML y recoge la informacion del mismo.
2. Envia dicha información al componente pintado en el layout.
3. El componente (Spinner, LIstView o GridView) pinta la información obtenida con su forma particuliar.

## Notas
He dejado los tres en ejecución para mayor rapidez viendo el funcionamiento del mismo.
